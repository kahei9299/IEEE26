# Demo Script (90s)

- 0-10s: Introduce SignCall Overlay and show the local video tile.
- 10-30s: Start signing a few sample phrases (HELLO, THANKS) and call out live captions.
- 30-50s: Trigger a correction with the button and show the acknowledgement caption.
- 50-70s: Explain the pipeline: frame capture -> landmarks -> classifier -> translator.
- 70-90s: Close with next steps (multi-user WebRTC, better models, UX polish).
